Rubrica Contatti in C#
=======================

Descrizione:
------------
Questa è una semplice applicazione console sviluppata in C# con .NET, che permette di:
- Aggiungere contatti
- Visualizzare contatti
- Cercare contatti
- Modificare contatti
- Eliminare contatti

I contatti vengono salvati automaticamente in un file JSON (`rubrica.json`), permettendo di mantenere i dati tra diverse sessioni.

Funzionalità:
-------------
- Gestione contatti (Create, Read, Update, Delete)
- Ricerca parziale per nome
- Gestione di contatti duplicati
- Ordinamento alfabetico automatico dei contatti
- Persistenza dei dati tramite file JSON

Tecnologie utilizzate:
-----------------------
- Linguaggio: C#
- Piattaforma: .NET 6/7/8
- IDE: Visual Studio 2022 Community Edition

Come avviare il progetto:
--------------------------
1. Aprire Visual Studio.
2. Creare un nuovo progetto "Console App" e sostituire il file `Program.cs` con il codice fornito.
3. Compilare ed eseguire il progetto.

Note:
-----
- Alla prima esecuzione, se il file `rubrica.json` non esiste, verrà creato automaticamente.
- Ogni modifica ai contatti viene salvata immediatamente nel file.
